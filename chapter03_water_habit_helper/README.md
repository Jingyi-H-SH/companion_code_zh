# 饮水习惯助手

这一章展示如何根据用户状态实现个性化 LLM 输出。它先预测饮水风险，再检索合适的提醒策略，最后生成定制化提醒。

## 工作流程

1. 从饮水日志中构建可解释的用户状态特征。
2. 让 LLM 基于已知特征估计当天未完成目标的风险。
3. 从本地提醒策略手册中检索匹配的策略片段。
4. 生成个性化提醒、一条微行动建议和简短解释。
5. 最后评估并可视化结果。

## 核心技术概念

- 基于用户状态的 Prompt 设计
- 把预测和生成放在同一条工作流里
- 把 RAG 用于“策略选择”，而不只是长文本问答
- 面向读者的个性化生成

## 如何运行

1. 在终端中进入当前章节文件夹。
2. 运行 `python run_demo.py`。
3. 如果环境里还没有 `OPENAI_API_KEY`，脚本会提示你输入。
4. 运行结束后，查看 `outputs/`。
